# Перемикач теми (світла/темна) для проєкту

✅ Готовий перемикач (світла/темна тема)  
✅ Зберігає вибір у localStorage  
✅ Додає клас `dark-theme` на <body>

## Як користуватись:

У своїх стилях додавай `body.dark-theme`:

```css - приклад
/* Світла тема */
.section {
  background: #fff;
  color: #000;
}

/* Темна тема */
body.dark-theme .section {
  background: #2a2d32;
  color: #fff;
}
```
